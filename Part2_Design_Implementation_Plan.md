# Part 2 (Session 2) Design Implementation Plan

---

## 1. Slide Density Targets

**Goal:** Achieve 85%+ slide area utilization

### Strategy:
- **Font sizes:** Use 10pt as PRIMARY body text (65% of all text)
  - 10pt: Descriptions, body text, detailed bullet points
  - 12pt: Main bullet points, emphasis text (20-25%)
  - 14pt: Section headers, large bullets (6%)
  - 16pt Bold: Governing messages
  - 20pt Bold: Slide titles

- **Content per slide:**
  - Maximum information density without cramming
  - Use 2-column layouts for comparisons
  - Use tables for structured data (4×4 matrix, scorecards)
  - Minimize whitespace - use full slide width/height

- **Visual elements:**
  - 20-50 shapes per slide (minimum 20)
  - Flowcharts, process diagrams, arrows
  - Rectangle boxes wrapping all text
  - Comparison matrices (before/after, A vs B)

---

## 2. Shape Usage Plan

### Total shapes per slide: MINIMUM 20 shapes

### Shape types and purposes:

#### Rectangles / AUTO_SHAPES (30-40% of shapes)
- Wrap ALL text content (no floating text!)
- Background boxes for sections
- Highlighting key concepts
- Creating grid structures

#### GROUPS (70-80% of shapes must be grouped!)
- Group related elements: process step group, comparison box group
- Timeline phase group (for sequential content)
- Matrix cell group (for Kraljic-style 2×2 tables)
- Icon + text groups

#### Arrows / Connectors (10-15% of shapes)
- Show time sequence: Multi-Sourcing → Dual Sourcing → Long-term Agreement
- Process flows: RFQ → Competitive Bidding → TCO Analysis → Supplier Selection
- Relationships: Bottleneck strategy → Supply stability
- Comparisons: Traditional approach → Modern approach

#### Triangles / Indicators (5% of shapes)
- Priority indicators (high/low)
- Increase/decrease trends
- Risk levels

## 4. Toy Page Layout Implementation

**Slides using Toy Page pattern:** 6, 9, 10 (3 slides)

### Structure:
- **Left side (60-70% width):** Visual diagrams, flowcharts, process flows
  - Position: x = 0.8", width = ~6.5-7.5"
  - Content: Diagrams, timelines with arrows, comparison matrices

- **Right side (30-40% width):** Text explanations
  - Position: x = ~7.5-8.0", width = ~2.5-3.0"
  - Content: 시사점 (Implications), 방안 (Approaches), Key takeaways
  - Font: 10pt for descriptions, 12pt for bullets

### Example (Slide 6):
```
Left (60%):
  [Diagram: Main Supplier → Backup Supplier structure]
  [Geographic distribution map]
  [Technology transfer flow]

Right (40%):
  시사점:
  - Single source risk reduction
  - Regional diversification benefits

  방안:
  - Dual sourcing implementation
  - Alternative material development
```
