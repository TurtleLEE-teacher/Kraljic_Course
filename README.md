# 전략적 재고운영 및 자재계획수립 과정

> **Kraljic Matrix 기반 전략적 재고관리 및 자재계획 실무 교육**

## 📚 과정 개요

이 레포지토리는 **Kraljic Matrix를 활용한 전략적 재고운영 및 자재계획수립**에 대한 종합 교육 자료를 담고 있습니다. Just-In-Time(JIT)에서 Just-In-Case(JIC)로의 패러다임 전환을 다루며, 실무에 바로 적용 가능한 프레임워크와 사례를 제공합니다.

### 🎯 학습 목표

- Kraljic Matrix를 활용한 자재 포트폴리오 분류 및 관리 전략 수립
- 자재군별(전략/레버리지/병목/일상) 최적화된 계획 방법론 습득
- ABC-XYZ 분석을 통한 재고 세분화 및 운영 전략 수립
- 공급업체 성과 평가 및 관계 관리 역량 강화
- ROP, MRP, 하이브리드 계획 등 자재계획 방법론 실무 적용

## 🎨 Kraljic Matrix Framework

### 4가지 자재군 분류

```
              높음 ↑ 구매 금액
                    │
    ┌───────────────┼───────────────┐
    │               │               │
    │  전략자재     │  레버리지자재 │
    │  (Strategic)  │  (Leverage)   │
    │               │               │
    ├───────────────┼───────────────┤
    │               │               │
    │  병목자재     │  일상자재     │
    │  (Bottleneck) │  (Routine)    │
    │               │               │
    └───────────────┼───────────────┘
                    │
         낮음 ← 공급 리스크 → 높음
```

### 자재군별 특징 및 전략

| 자재군 | 공급 리스크 | 구매 금액 | 주요 전략 | 계획 방법 |
|--------|:-----------:|:---------:|----------|-----------|
| **전략자재** | 높음 | 높음 | 장기 파트너십, 협력적 관계 | 하이브리드 계획 |
| **레버리지자재** | 낮음 | 높음 | 경쟁입찰, 볼륨 레버리지 | MRP |
| **병목자재** | 높음 | 낮음 | 공급 안정성 확보, 안전재고 | ROP |
| **일상자재** | 낮음 | 낮음 | 프로세스 효율화, 자동화 | 자동화 시스템 |
